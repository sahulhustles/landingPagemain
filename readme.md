## ğŸš€ Getting Started

### Prerequisites
- A modern web browser
- Local web server (optional, for local development)

### Installation

1. **Download/Clone the project**
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server for better development experience

3. **Using a local server (recommended)**
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

4. **Access the website**
   - Open `http://localhost:8000` in your browser

## ğŸ”§ Development

### File Organization
- Keep HTML files in root directory
- CSS in `/css/` folder
- JavaScript in `/js/` folder
- Images should be placed in `/images/` folder (create as needed)

### Adding New Pages
1. Create new HTML file in root directory
2. Copy navigation structure from existing pages
3. Update navigation links in all HTML files
4. Add new page link to `script.js` for active state management

### Performance Tips
- Images should be optimized (WebP format recommended)
- Use lazy loading for images below the fold
- Minify CSS and JavaScript for production
- Enable gzip compression on server
